# 赛马娘同人文写作风格调研报告

> 调研日期：2026-05-19 | 数据来源：话本小说、百合文库、B站、知乎、NGA、ハーメルン、pixiv（间接）、刺猬猫（间接）

---

## 一、三大圈层的生态概览

### 1.1 中文圈（刺猬猫 / 话本 / 飞卢 / 菠萝包 / Lofter / B站）

**核心平台分化明显：**

| 平台 | 主流风格 | 篇幅偏好 |
|------|----------|----------|
| 刺猬猫 | 穿越/系统/马娘训练员题材，中长篇连载 | 50~300章 |
| 话本小说 | CP向/日常向/怪文书，短篇为主 | 1~50章 |
| 飞卢 | 系统流/无敌流/后宫向 | 100~500章 |
| Lofter | 短篇/片段/CP向/虐文/甜文 | 1k~10k字 |
| B站专栏 | 怪文书/视频配套短篇/同人设定集 | 不定 |

**中文圈最显著的特征：** 穿越（转生）是绝对主流入口。主角几乎都是穿越到特雷森学园成为训练员/变成马娘/成为校工/成为马主，带系统或带记忆重生。

### 1.2 日文圈（pixiv小説 / ハーメルン / 小説家になろう）

**日文圈的主阵地：**

- **ハーメルン（hameln）**：赛马娘同人小说的最大聚集地，分数制和评论文化成熟。高评分作品如「リィンカーネーションダービー」（转生训练员+最弱马娘逆袭，hameln 9.6分）
- **pixiv小説**：短篇为主，CP向和「もしも」（if）系列极为繁荣，tag系统精细
- **小説家になろう**：偏向一般向/异世界融合，数量不如hameln

**日文圈的核心特征：**
- 训练员 x 马娘 CP 是最大类型
- 「歴史改変IF」——基于史实但改写关键比赛结果的if线，成熟且受众固定
- 日常系和「怪文書」（类似中文圈怪文书但更偏向黑色幽默和角色崩坏）两极化
- pixiv上「ウマ娘 夢小説」（梦小说=读者代入型）有固定受众

### 1.3 英文圈（AO3 / FanFiction.net / Reddit）

**规模最小，但结构稳定：**
- AO3上 Umamusume: Pretty Derby 的tag约有2000+作品（远少于日文和中文）
- 英文圈更偏向角色分析和关系研究，而非竞技描写
- 常见的tag：Alternate Universe - Canon Divergence（历史分支）、Fluff（甜文）、Angst（虐文）、Hurt/Comfort
- 写作风格更接近欧美同人圈传统——注重心理描写和角色关系张力
- **Trainer/OC x Canon Character**（原创训练员与原作角色CP）占主流
- 英文圈读者对日本赛马史实的了解较少，作品偏向借用角色设定进行原创故事

## 二、热门同人类型图谱

根据收集到的实际作品标题/内容进行分类：

### 2.1 穿越·转生类（中文圈最主流）

**代表作品：**
- 「穿越赛马娘：疾驰的蝶影」「穿越赛马娘」「从零开始的赛马娘世界」
- 「关于我成为赛马这件事」（471章连载中）
- 「宅男穿越到赛马娘世界」

**细分方向：**
| 子类型 | 内容 | 参考作品 |
|--------|------|----------|
| 穿越成训练员 | 现代人穿越成特雷森训练员，用前世知识培养马娘 | 「リィンカーネーションダービー」 |
| 穿越成马娘 | 魂穿成某位马娘，参与比赛改写历史 | 「关于我成为赛马这件事」 |
| 穿越成马主/骑手 | 经营牧场/训练，偏经济经营 | 「我在东京养赛马」 |
| 系统穿越 | 带金手指系统穿越，成长数值化 | 「穿越赛马娘：寿命仅剩30天」 |

**写作特征：**
- 开局必有「醒来发现穿越了」的经典桥段
- 利用前世知识（赛马知识/历史结果）创造信息差
- 叙事节奏快，爽点密集，适合中长篇网文化连载

### 2.2 历史IF线·史实改写类（中日共通）

**核心魅力：** 读者知道真实历史的结果，作者构建一个「如果当时……」的新可能。

**常见改写节点：**
- 无声铃鹿第4次骨折 —— 如果当时没有受伤
- 东海帝王第三次骨折 —— 如果帝王以完全健康的状态参战
- 好歌剧的七冠之路 —— 如果在某个关键比赛中翻盘
- 春丽（乌拉拉）的首胜 —— 给予这个从未赢过的马娘一场胜利
- 小栗帽的传说 —— 改写某个微妙的名次

**代表类型：**
- 「如果那个女孩没有去特雷森学园的话」——《小特前辈的场合》（B站热门）
- 「当小栗帽赢下了黄金青年杯」—— 史实分支，讲述改变后的命运
- 「无败十冠」—— 让某位马娘达成史实中没有的全胜战绩

**写作特征：**
- 需要扎实的史实功底，作者必须懂赛马知识（至少了解该马娘的真实赛绩）
- 情感张力来自「读者知道的悲剧被扭转」
- 需要处理好「历史之壁」——完全架空会让老粉失去共鸣

### 2.3 CP向·恋爱类（全平台通吃）

**热门CP类型：**
- 训练员 x 特定马娘（全平台最主流，角色选择影响读者喜好）
- 马娘 x 马娘（pixiv上大量百合向作品，AO3上也有）
- 原创角色 x 马娘（中文网文主流）

**写作特征：**
- 日常互动 > 比赛描写（比赛只是背景）
- 常见桥段：因训练接触→互相理解→羁绊加深→感情爆发
- 怪文书经常出现黑化/独占/病娇等极端情感表现

### 2.4 怪文书·黑暗系（日中文圈特有现象）

「怪文书」是赛马娘同人社区独有的亚文化，特点是：
- 角色性格扭曲/黑化/极致化（基于原作性格的极端延伸）
- 黑色幽默与致郁并存
- 伦理挑战（囚禁/精神控制/暴力关系）
- 以荒诞和夸张投射原作中的压抑要素

**典型作品链：**
- 「帝下室」系列——东海帝王被囚禁/黑化的暗黑故事（B站播放量极高）
- 「黑暗特雷森学院」系列——将学园设定成黑暗组织的AU
- 「初生系列」（初生の寄寄子、初生对内恰的离婚协议等）
- 「繁育场宣传海报」——极度黑暗、刻意冒犯性内容

**写作特征：**
- 文风极端化，刻意触碰禁忌
- 在中文Lofter/B站/话本里自成圈子
- 读者群体有明确「预警文化」（标题/简介里标注【黑暗慎入】）
- 本质是「用极端情境探索角色深处」的严肃文学尝试，但被包裹在荒诞外衣下

### 2.5 日常·轻喜剧类（全平台基石）

**代表作品：**
- 「赛马娘：甜蜜的约定」「赛马娘的想象」
- 「训练员与天空」（星云天空CP日常）
- 「赛马娘：特雷森大闹吉庆岛」

**写作特征：**
- 轻小说笔法，对话为主
- 角色性格还原度高（粉丝看重「不OOC」）
- 比赛只是甜品，重点是马娘们的互动和萌点

### 2.6 推理·类型文学类（小众但精）

**代表作品：**
- 「赛马娘推理文」系列（百合文库有04/07等多篇）
- 「赛马娘新编」——重写经典故事

**写作特征：**
- 借赛马娘皮囊写类型文学（推理/悬疑/冒险）
- 需要巧妙的设定自洽
- 对普通读者门槛较高，但容易收获极高评价

### 2.7 跨界融合·乱入类（中文网文特色）

**代表作品：**
- 「赛马娘之假面骑士SPIRITS」「假面骑士极狐之赛马娘」
- 「赛马娘小马宝莉客串」「赛马娘：替嫁被鲁道夫休后，我成了丸善斯基的妻子」
- 「为了赚军费！拼了！」——赛马娘x军事主题

**写作特征：**
- 不同IP的碰撞是核心卖点
- 结构松散，世界观融合方式随性
- 受众明确——两边IP都懂的读者

## 三、写作风格分析

### 3.1 中文网文主流风格

**特点：**
- 开篇黄金三章原则，快切入快展开
- 大量对话推进剧情
- 适当系统面板/数值化（刺猬猫/飞卢常见）
- 章节末尾留钩子（cliffhanger）
- 每章2000~4000字为主，段落短，适合手机阅读

**缺点：**
- 容易陷入训练→比赛→训练→比赛的循环
- 中期节奏容易拖沓（流水账化，如《关于我成为赛马这件事》的评论反馈）
- 历史考据偏少，硬核粉丝可能出戏

### 3.2 日文同人风格

**特点：**
- 注重氛围和角色内心世界
- 比赛描写细腻，有真实赛马转播风格的文字渲染
- 「歴史改変IF」类注重史料考据
- 叙事节奏较慢，让情感充分发酵
- 对话框形式（台本形式）在pixiv短篇中常见

**代表方法：**
- 用转播式第三人称写比赛：「第4コーナーを回り、残すは直線200m——」
- 搭配解说员风格的激情叙述
- 马娘的内心独白与比赛实况交错推进

### 3.3 怪文书/极端风格

**特点：**
- 第一人称或局限性第三人称
- 语言富有攻击性和冲击力
- 短句，短段落，刻意断裂的节奏
- 用大量留白制造窒息感

### 3.4 英文圈同人风格

**特点：**
- 第三人称有限视角（紧贴一个角色的意识流）
- 注重心理描写和情感递进
- 常见「5+1 Things」式结构（五个场景+一个高潮）
- 慢热，铺垫长，情感爆发在末尾

## 四、爽点构成体系

赛马娘同人的爽点可以拆解为以下六大类型，按出现频率排序：

### 4.1 逆袭爽（最大公约数）

**机制：** 弱者（弱马娘/新人训练员）→ 通过努力和策略 → 击败强者

**应用场景：**
- 春丽/乌拉拉终于赢得比赛
- 最弱马娘在训练员培养下夺得G1
- 新人训练员证明自己

**为什么爽：** 赛马娘企划本身的母题就是「在历史上留下名字」，逆袭是刻在IP基因里的叙事驱动。再加上许多马娘（如乌拉拉/米浴/双涡轮）的史实本身就是悲壮逆袭（或逆袭失败），改写或还原这种逆袭能直接触发情感共鸣。

### 4.2 历史改写爽（核心特色）

**机制：** 读者知道「真实结果」→ 故事给出「不同结果」→ 释放被历史压抑的情感

**关键场景：**
- 无声铃鹿没有受伤，继续奔跑
- 东海帝王没有骨折，无败三冠
- 米浴没有在比赛中出事
- 好歌剧被正名（史实中七冠但被轻视）

**为什么爽：** 这是赛马娘同人**独有的**爽点类型。它依赖于现实赛马的悲剧性。粉丝对马娘的感情有一部分来自「替历史中的那匹马遗憾」，同人提供了解放这种遗憾的出口。

### 4.3 养成·羁绊爽

**机制：** 训练员与马娘从陌生 → 信任 → 共同奋斗 → 闪耀时刻

**应用场景：**
- 首次共同拿下重赏的泪目瞬间
- 训练员为马娘做出牺牲
- 退役/引退前的最后一战

**为什么爽：** 这直接对应游戏的核心玩法（养成+剧情解锁），读者对「训练员与马娘的羁绊」有天然的情感投资意愿。

### 4.4 竞赛临场爽

**机制：** 赛道上的紧张拉锯 → 逆转或惜败 → 情绪释放

**关键要素：**
- 逼真的比赛过程描写（步速/位置/呼吸/观众反应）
- 实况解说式的节奏控制
- 最后200米冲刺的爆发描写

**为什么爽：** 纯竞技层面的刺激。好的比赛描写能让读者心跳加速，即使知道结果也像看直播一样紧张。

### 4.5 知识碾压爽（中文穿越类特有）

**机制：** 穿越者利用现代知识/赛马历史知识 → 在特雷森世界建立优势

**应用场景：**
- 训练员知道某马娘的潜力，提前签约
- 利用马主时代的知识做出正确决策
- 用现代训练科学优化训练计划

**为什么爽：** 信息和认知的优势本身就是爽点。读者和主角共享「知道历史」的上帝视角，产生认同感。

### 4.6 情感爆发爽（全品类通吃）

**机制：** 长期压抑/误会/暗恋 → 引爆时刻

**应用场景：**
- 赛后拥抱哭诉
- 误会的解除
- 训练员的告白（或被马娘告白）
- 退役式上的感人致辞

**为什么爽：** 情感积累后的释放是最原始的爽点。赛马娘「奔跑+少女」的设定天然适合情感爆发。

## 五、推荐写作套路

### 5.1 黄金公式：穿越+养成+改写（中文网文向）

**结构模板：**
1. **第1~3章**：穿越/觉醒，认识第一位马娘，建立初步目标
2. **第4~10章**：第一次训练/比赛（小赛），展示系统/金手指
3. **第11~20章**：建立队伍，第二主角马娘加入
4. **第21~30章**：第一次重赏挑战，成功或惜败
5. **第31~50章**：G1级大赛，高燃比赛+情感爆发
6. **第51+**：持续循环（新马娘+新赛事+史实改写），加入危机与转机

**适合平台：** 刺猬猫、飞卢、菠萝包
**注意事项：** 注意节奏变化，每10章左右安排一场有分量的比赛；避免训练日常流水账；中期加入冲突（其他训练员的竞争/马娘的伤病/学园危机）

### 5.2 历史IF线（品质向/硬核向）

**结构模板：**
1. **序章（1章）**：锚定史实分歧点，一句话或一段话说明「如果当时……？」
2. **发展期（3~5章）**：日常训练+角色关系+少量比赛，铺垫新走向
3. **关键节点（1~2章）**：分歧点的正式改写，高燃比赛
4. **后果期（3~5章）**：改写后的连锁反应（蝴蝶效应）
5. **结局（1章）**：新历史下的谢幕演出

**核心技巧：**
- 开头简明扼要地点出「历史之壁」的位置
- 对真实赛马历史的考据越扎实越好（粉丝会查）
- 改写要付出代价（不能无代价地改写——否则失去张力）
- 用原史实的悲剧衬托新路线的温暖

**适合平台：** Lofter（中短篇）、pixiv、B站专栏

### 5.3 训练员x马娘CP长篇（情感向）

**结构模板：**
1. **相遇篇（5~8章）**：分配/偶遇，初期的抗拒或好奇
2. **磨合篇（8~15章）**：训练中的摩擦与理解，第一次小比赛
3. **信赖篇（10~20章）**：突破瓶颈/战胜强敌，关系升温
4. **考验篇（10~15章）**：伤病/低谷/第三者，CP面临危机
5. **闪耀篇（5~10章）**：夺冠/告白/新起点

**核心技巧：**
- 比赛描写聚焦训练员和马娘的共同视角（而非上帝视角）
- 情感推进要慢，不要急着发糖
- 用「训练细节」替代直接的心理描写（展示而非说教）

### 5.4 怪文书（实验向/亚文化向）

**结构特点：**
- 短小精悍，通常单章或三章以内
- 开篇直接丢出极端情境，不铺垫
- 全程高密度高压线行进
- 结尾留白，不给解释

**核心技巧：**
- 标题直接预警（【黑暗慎入】【R18G】【病娇】）
- 角色选择有讲究（东海帝王/米浴/爱丽速子等「有暗面」的角色更适合）
- 怪文书不是乱写，需要对角色有深刻理解

### 5.5 「比赛的写法」专项建议

比赛场面是赛马娘同人的核心技艺，三种写法各有优劣：

| 写法 | 优势 | 适合场景 |
|------|------|----------|
| **转播式**（第三人称实况解说） | 临场感最强，紧张刺激 | 重磅G1比赛 |
| **马娘内视角**（第一人称POV） | 情感深度，能写内心挣扎 | 逆袭战/关键转折 |
| **训练员外视角**（场边观察） | 既有距离感又有代入感 | CP向作品的比赛片段 |

**比赛节奏控制的经验法则：**
- 1200字以下的小场面：跳着写，只写关键转折点
- 1200~3000字的正式比赛：写出发→中盘→弯道→最后冲刺
- 3000字以上的核心大战：加入解说/观众反应/对手视角/慢镜头回忆

## 六、平台特点与发表策略

### 6.1 各平台推荐方向

| 平台 | 推荐创作方向 | 避坑 |
|------|-------------|------|
| **刺猬猫** | 穿越系统长篇、训练员x马娘 | 避免过于硬核的历史考据（受众偏网文化） |
| **Lofter** | 短篇CP向、怪文书、历史IF | 重视tag体系，注意预警标注 |
| **话本小说** | 日常向、非主流CP、中型连载 | 更新频率要求不高 |
| **飞卢** | 系统流、后宫向、升级流 | 节奏要快，每章必有进展 |
| **B站专栏** | 怪文书、设定集、科普向 | 注意审核尺度（怪文书容易被举报） |
| **pixiv小説** | 短篇、CP向、歴史改変IF | 日文写作，tweet风格（短句/台本）受欢迎 |
| **ハーメルン** | 中长篇日文同人 | 引用史实出典时标注来源，评分文化很重要 |

### 6.2 通用建议

1. **入门选择：** 从「穿越成训练员+一支队伍+一个核心改写目标」入手，结构清晰，容错率高
2. **进阶尝试：** 历史IF线——需要对某匹马娘的史实有深入理解，做完备的史料预习
3. **文笔追求：** 比赛描写是试金石，建议至少练好「转播式」和「内视角」两种写法
4. **节奏控制：** 比赛密度不要过高（一章比赛后至少2~3章日常/情感回），也不要过低（超过5章无比赛会流失追更）
5. **卖点明确：** 让读者在前三章就清楚「这篇的核心看点是什么」

### 6.3 观察到的流行趋势（2026年5月）

- **穿越+系统**仍然是中文圈最安全的类型，但同质化严重，需要在角色选择/改写方向上做出差异化
- **历史IF线**正在从短篇向中篇进化，硬核粉丝群体在增长（受动画RTTT和第三季影响）
- **怪文书**的分化加剧：纯黑化类型在退潮，「有病但温暖」的新型怪文书正在兴起
- **「变成马娘」类（性别转换/种族转换）** 在B站和话本持续增长
- **非主流马娘（菱钻奇宝/双涡轮/也文摄辉等）** 的主角文更容易获得口碑，卷的程度远低于特别周/东海帝王/无声铃鹿
